# TikTok Claims Classification — Exploratory Data Analysis (EDA)

This project analyzes a TikTok video dataset (dummy dataset) to explore the differences between **claim** and **opinion** content.  
The goal is to prepare the data for a future machine learning model by performing **data understanding, feature engineering, and anomaly detection**.

---

## 📌 Project Overview

- Examined the structure of TikTok video metadata  
- Analyzed the distribution of claim vs. opinion content  
- Explored engagement metrics (views, likes, shares, comments)  
- Investigated user attributes such as verification and ban status  
- Identified anomalies and outliers  
- Created new features (text_length, engagement_rate)  
- Conducted exploratory data analysis using Python and Pandas
